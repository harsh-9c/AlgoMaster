          <h3>PROBLEM:</h3>
          <p style="color: #989898; font-size: 18px">
            You are given a binary string SS of length NN. You can perform the
            following operation on SS: Pick any set of indices such that no two
            picked indices are adjacent. Flip the values at the picked indices
            (i.e. change 00 to 11 and 11 to 00). For example, consider the
            string SS = 11011011101101. Pick the indices
            \\\\\\\\\\\\\\\\\\\\\{1,3,6\}{1,3,6}. Flipping the values at picked
            indices, we get \underline{1}1\underline{0}11\underline{0}1â0111111
            1 â 1 0 â 11 0 â 1â0111111. Note that we cannot pick the set
            \{2,3,5\}{2,3,5} since 22 and 33 are adjacent indices. Find the
            minimum number of operations required to convert all the characters
            of SS to 00.
          </p>
          <h3>EXPLANATION:</h3>
          <p style="color: #989898; font-size: 18px">
            For each test case we have a binary string of length NN. Now as no
            adjacent bits can be flipped together. Three cases are possible:
            <ul style="color: #989898; font-size: 18px">
              <li>When the string is already 00, then the number of minimum
                operations will be zero.</li>
              <li>When no two adjacent bits are set. In this case all the set
                bits can be converted to 00 in 11 operation.</li>
              <li>When the string contains adjacent set bits. In this case it
                can be shown that the answer will be 22 since all the
                odd-indexed set bits and even-indexed set bits can be identified
                as two different sets.</li>
            </ul>
            Evaluating all the above possibilities gives us the result.
          </p>
          <h3>TIME COMPLEXITY:</h3>
          <p style="color: #989898; font-size: 18px">O(N) for each test case.</p>
